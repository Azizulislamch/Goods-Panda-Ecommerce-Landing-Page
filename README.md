# 🐼 Goods Panda - Ecommerce Landing Page

A modern and responsive **E-commerce Landing Page** built using **HTML5 and CSS3**.  
This project showcases a clean UI design featuring product sections, categories, and a newsletter subscription area.

🔗 **Live Demo:** https://azizulislamch.github.io/Goods-Panda-Ecommerce-Landing-Page/  
📂 **Repository:** https://github.com/Azizulislamch/Goods-Panda-Ecommerce-Landing-Page

---

# 📌 Project Overview

Goods Panda is a simple yet elegant **e-commerce landing page** designed to display featured products and categories in a visually appealing layout.

The landing page includes:

- Hero section with featured product
- Product categories
- Shoes product section
- Backpack product section
- Newsletter subscription section
- Fully responsive layout

This project was built to practice **responsive web design and modern UI layout using Flexbox.**

# 📱 Responsive Design

This project is fully responsive and optimized for different devices.

| Device | Breakpoint |
|------|------|
| Laptop | max-width: 1200px |
| Tablet | max-width: 992px |
| Mobile | max-width: 640px |
| Small Devices | up to 320px |

Responsive adjustments include:

- Hero section stacking on smaller screens
- Categories wrapping on tablet devices
- Product cards stacking vertically
- Newsletter input and button aligning vertically on mobile
